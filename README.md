## Install

1. Clone the project :
``` Shell
git clone https://github.com/fjulien/2048-game.git
```
2. Install project :
``` Shell
npm install
```
3. Launch the project :
``` Shell
npm start
```
Automatically the project starts in this url => [http://localhost:3000](http://localhost:3000)

## Lauch tests

``` Shell
npm run test
```
